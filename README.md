# 🏦 ATM Banking System

A simple ATM Banking System project that simulates basic banking operations like withdrawal, deposit, and balance inquiry. This project demonstrates core programming concepts and mimics real-world ATM functionality.

---

## 📌 Project Overview

This project simulates an Automated Teller Machine (ATM) where users can perform essential banking operations through a simple interface.

---

## 🚀 Features

- 🔐 User authentication (PIN / credentials)
- 💰 Balance inquiry
- 💵 Cash withdrawal
- 💳 Deposit functionality
- 🧾 Basic transaction handling
